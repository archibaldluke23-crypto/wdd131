Film Quiz Site Plan

Luke Archibald

WDD 131.06

Overview

Purpose

A movie quiz containing all my favorite movies which requires the user to guess which movie best fits the description. I hope to create a fun trivia experience as well as give movie recommendations to the users.

Audience

This app is made for casual and diehard film enjoyers

Dynamic elements

I will use a function to open images of the correct film when they select an answer. I will have all the images in a hide class and will unhide them when one of the buttons detects a click. I will then see what value that button has and compare it to the correct answer button and if it matches I will highlight the button to be green and display correct. If it is incorrect I will highlight the clicked button red and highlight the correct answer green as well as display incorrect. I will also use and array to keep track of any movies they got wrong and tell them to watch or rewatch those movies. I will add objects to the array to keep track of the movie they got wrong as well as the image to display at the end of the page.
